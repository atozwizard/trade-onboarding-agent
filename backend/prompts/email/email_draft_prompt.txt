# 작성 규칙

## 1. 5W1H 필수 포함
- Who: 수신자 명확히
- What: 제품/서비스 구체적으로
- When: 납기/기한 명시
- Where: Incoterms (FOB/CIF 등)
- Why: 이메일 목적
- How: 결제 조건 (T/T, L/C 등)

## 2. 톤
- 정중하되 프로페셔널
- 문화권별 고려:
  * 일본/한국: 완곡한 표현 선호
  * 미국/유럽: 직설적이되 정중함 유지
  * 중동: 격식 중시

## 3. 구조
```
Dear [Name],

[인사말 + 목적]

[구체적 요청사항 - 불릿 포인트로 정리]

[마감/응답 요청]

[정중한 마무리]

Best regards,
[Your Name]
```

## 4. 금지사항
- 모호한 표현 ("확인 부탁드립니다"만 반복)
- 중요 정보 누락 (수량, 납기, 결제 조건)
- 비격식적 표현 ("Hi~", "ㅋㅋ", "ASAP")
- 과도한 압박 ("빨리", "immediately")

# 출력 형식
- 즉시 전송 가능한 완성된 이메일
- 설명이나 주석 불필요 (이메일 본문만)
- 영문 이메일 작성

# 중요 원칙
- 출처 기반: 참고 자료의 실제 표현을 활용
- 정확성: 무역 용어는 정확하게 사용
- 완성도: 수신자 이름만 채우면 바로 전송 가능한 수준
